# Snowflake Cortex AI SQL: Extracting Insights from Multimodal Customer Service Data

## Overview

In this quickstart, you'll learn how to build a comprehensive customer service analytics system that processes audio, text, and document data using Snowflake Cortex AI functions. This application demonstrates how to extract insights from multimodal data sources including call recordings, chat logs, support tickets, and PDF documents using powerful AI capabilities like transcription, translation, sentiment analysis, classification, and intelligent summarization. By the end of this session, you will have a clear understanding of how to use Cortex AI functions to analyze customer service data at scale, all while keeping your data secure within Snowflake's environment.
